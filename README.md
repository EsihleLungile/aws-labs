# ☁️ AWS Cloud Labs

Hands-on AWS labs, configurations, and walkthroughs documenting my practical cloud engineering journey.

Each folder contains a real exercise I've worked through — including setup steps, configurations used, and key takeaways.

---

## 📁 Structure

| Folder | Description |
|---|---|
| [`/ec2`](./ec2/) | EC2 instance setup, SSH access, and configuration |
| [`/vpc`](./vpc/) | VPC design — subnets, route tables, internet gateways |
| [`/iam`](./iam/) | IAM users, roles, policies, and best practices |
| [`/cloudwatch`](./cloudwatch/) | Monitoring, alarms, and log groups |
